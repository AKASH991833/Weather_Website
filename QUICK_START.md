# ⚡ Quick Start Guide

## 3 Steps to Deploy

### 1️⃣ Add Your API Key
File: `js/config.js`
```javascript
API_KEY: 'YOUR_API_KEY_HERE'  // ← Replace with your real key
```

Get a free key: https://openweathermap.org/api

### 2️⃣ Generate Icons (Optional)
Open `generate-icons.html` in your browser → Click "Generate Icons" → Download ZIP → Extract to `icons/`

### 3️⃣ Push to GitHub
```bash
git add .
git commit -m "Deploy weather app"
git push
```

Then enable GitHub Pages in your repository settings.

---

## That's it! 🎉

Your site will be live at: `https://YOUR_USERNAME.github.io/YOUR_REPO/`

For detailed instructions, see [GITHUB_PAGES_DEPLOYMENT.md](./GITHUB_PAGES_DEPLOYMENT.md)
